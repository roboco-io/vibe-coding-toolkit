# 디버깅을 위한 Tidy First 프롬프트

## 상황 설정
다음 코드에서 [구체적인 문제 설명]이 발생하고 있습니다.

## 요청사항
Tidy First 원칙에 따라 다음 순서로 문제를 해결해주세요:

1. **코드 정리 단계**
   - 보호 구문 추가로 예외 상황 처리
   - 복잡한 조건문을 설명하는 변수로 분리
   - 긴 함수를 작은 단위로 분해

2. **디버깅 단계**
   - 문제의 근본 원인 파악
   - 안전한 방법으로 수정
   - 재발 방지 방안 제시

3. **검증 단계**
   - 수정 후 코드의 동작 확인
   - 테스트 케이스 제안
